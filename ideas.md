**Core Principles:**
- Strict grid system with generous whitespace
- Typography-first hierarchy: size and weight convey structure
- Restrained color — deep navy as primary, cool gray for secondary, single gold accent
- Content-driven layout: no decorative elements that don't serve information

**Color Philosophy:** Deep navy (#0f2044) as the anchor, white for content areas, slate-gray (#64748b) for secondary text, gold (#c9a84c) for interactive highlights. Communicates precision, academic rigor, and professionalism — ideal for an engineering portfolio.

**Layout Paradigm:** Asymmetric two-column layouts with a narrow left column for labels/metadata and a wide right column for content. Navigation is a slim left sidebar on desktop.

**Signature Elements:** Thin horizontal rules as section dividers, small all-caps labels in gold above section titles, generous line-height in body text.

**Interaction Philosophy:** Smooth scroll-triggered reveals, subtle hover states with color transitions, no gimmicks.

**Animation:** Staggered fade-in on scroll entry, smooth page transitions via opacity.

**Typography System:** Playfair Display (display headings, italic for emphasis) + DM Sans (body, UI labels) — contrast between editorial serif and clean sans-serif.

## Selected Design: Swiss International Typographic Style

**Rationale:** The Swiss grid approach best serves an academic/professional portfolio. The Playfair Display + DM Sans pairing creates an editorial, distinguished feel that stands apart from generic tech portfolios. The deep navy + gold palette communicates Georgia Tech's academic prestige and Sidhaarth's engineering focus without resorting to clichéd "dark hacker" aesthetics.
